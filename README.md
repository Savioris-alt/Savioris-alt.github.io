🏆 177 RAW Tournament Bracket

Welcome to the tournament bracket website for the 177 RAW Discord server! Here you can track matches, results, and participant progress.

📌 How to Use

Open the bracket – the homepage shows the current tournament bracket.

Follow the matches – results are updated automatically as matches are completed.

Update results – if you notice an error or your match is finished, notify a server moderator.

🗂 Tournament Structure

Participants compete individually or in teams.

Matches follow the bracket structure.

Winners of each round advance further in the tournament.

📢 Rules

All participants must follow the 177 RAW server rules.

Fair play is mandatory.

Any disputes are resolved by server moderators.
